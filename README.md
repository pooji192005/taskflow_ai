# 🚀 TaskFlow AI - Intelligent Voice Automation System

TaskFlow AI is an AI-powered desktop assistant that understands natural language commands and performs tasks such as automation, code generation, and system operations.

---

## 🧠 Overview

TaskFlow AI is designed as an intelligent agent system that:
- Understands user commands using Natural Language Processing (NLP)
- Converts them into structured actions
- Executes tasks dynamically using modular components

The system follows a scalable architecture with planner, executor, and AI engine.

---

## ✨ Features

- Voice-based command execution
- Natural Language Processing (NLP)
- Task automation (open apps, web search, etc.)
- Code generation using AI
- Modular architecture (agents, planner, executor)
- File and system operations
- Email automation (requires API setup)

---

## 🛠️ Tech Stack

- Python
- Natural Language Processing (NLP)
- Speech Recognition
- Ollama (Local LLM)
- OpenAI API (optional)

## ⚙️ How to Run

### 1. Clone the repository
git clone https://github.com/pooji192005/taskflow_ai  
cd taskflow_ai  

### 2. Install dependencies
pip install -r requirements.txt  

### 3. Run the application
python main.py

## 🤖 AI Setup (Ollama)

Make sure Ollama is installed and running:

ollama run llama3
